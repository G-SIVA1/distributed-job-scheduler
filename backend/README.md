# Distributed Job Scheduler

## Features

- User Authentication
- Project Management
- Queue Management
- Job Scheduling
- Worker Management
- Retry Policy
- Dead Letter Queue
- Metrics API

## Tech Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- JWT
- Docker

## Run

pip install -r requirements.txt

python -m uvicorn app.main:app --reload

Swagger

http://127.0.0.1:8000/docs
